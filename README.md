# Pace Theme

An elegant VS Code theme optimized for developers with a high pace.

![Banner](./banner.jpg)

## Installation

1. Open the extensions sidebar in VS Code
2. Enter `Pace Theme` in the search
3. Select the theme and click on `Install`
